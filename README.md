# Coffee Sales Dashboard (Interactive Excel Project) ☕

## Project Overview
This project is a dynamic, single-page dashboard created in Microsoft Excel to analyze coffee sales data. The goal was to transform raw sales records into a visual tool that allows for quick decision-making and trend analysis across different regions and product types.

## Key Features & Functionality
* **Dynamic Data Retrieval:** Built using advanced `INDEX` and `MATCH` functions to link orders with product and customer data across multiple sheets.
* **Interactive Slicers:** Includes interactive filters for **Size**, **Roast Type Name**, and **Country**, allowing users to drill down into specific market segments.
* **Timeline Integration:** A visual timeline filter to navigate sales performance from 2019 through early 2020.
* **Visual Analytics:** * **Total Sales Over Time:** A multi-line chart comparing the performance of Arabica, Excelsa, Liberica, and Robusta beans.
  * **Geographic Distribution:** A bar chart showcasing sales volume by country (United States, Ireland, and United Kingdom).

## Technical Skills Applied
* **Advanced Formulas:** `INDEX-MATCH` for complex lookups.
* **Data Visualization:** Custom formatting of line and bar charts for a professional UI/UX.
* **Pivot Tables:** Used to aggregate data for the charts.
* **Dashboard Design:** Created a "Dark Mode" aesthetic with a focus on readability and clean layout.

## How to Use
1. Download the `.xlsx` file from this repository.
2. Open in Microsoft Excel.
3. Use the **Slicers** on the right and the **Timeline** at the top to interact with the charts.
